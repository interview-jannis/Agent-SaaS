## 1. Admin — Products 가격 표시 개선

### 문제
KRW/USD 구분 없이 무조건 원화(₩)로 표시되던 버그

### 해결
- `products` 테이블에 `price_currency` 컬럼 추가
  ```sql
  ALTER TABLE products ADD COLUMN IF NOT EXISTS price_currency TEXT NOT NULL DEFAULT 'KRW';
  ```
- 가격 입력 UI → 단일 input + KRW/USD 토글 버튼으로 변경
- 3자리 쉼표 자동 포맷팅 (`fmtNum()`)
- 입력 중 실시간 환산 힌트 표시 (예: `₩1,350,000 ≈ $1,000`)

### 아키텍처 결정
> DB에는 입력한 값 그대로 저장 + `price_currency`로 통화 구분
> Agent 화면 → 무조건 USD 환산 표시
> Admin 화면 → 입력 통화 그대로 표시 (KRW면 ₩, USD면 $)

## 4. 세션 오류 처리 (`SessionGuard`)

### 문제
```
AuthApiError: Invalid Refresh Token: Refresh Token Not Found
```
- Supabase access token(1시간) 만료 시 refresh token으로 자동 갱신 시도
- Refresh token이 이미 rotate되었거나 만료되면 갱신 실패 → `SIGNED_OUT` 이벤트 발생
- 아무도 이벤트를 듣지 않아 에러만 콘솔에 찍히고 앱 먹통

### 해결
`SessionGuard` 컴포넌트 생성 (`src/components/SessionGuard.tsx`)
```tsx
supabase.auth.onAuthStateChange((event) => {
  if (event === 'SIGNED_OUT') router.replace('/login')
})
```
- `/agent/layout.tsx`, `/admin/layout.tsx` 양쪽에 추가
- 즉각 처리: 브라우저 LocalStorage에서 `sb-*` 키 삭제 후 재로그인
