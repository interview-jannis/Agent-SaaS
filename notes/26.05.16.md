   - `+` 4개+ paren-aware 분할 (paren 안 `+`는 보존)
   - 인라인 `•` / `·` (middle dot) 자동 분할 (Health Screening 항목)

4. **오탈자 일괄 fix**: `Clinc → Clinic`, `[hest → Chest`, `SPick → Pick`, `Assessmen → Assessment`, `Custermize → Customize`, `VP Premium → VIP Premium`, `Uninalysis → Urinalysis`, `Creatnine → Creatinine`, `Primier → Premier`, `Coner → Corner`, `Seul → Seoul`, `Trevel → Travel`, `Gwangak → Gwanak`
